Shadowsocks-Android新版本kcptun插件设置方法

发表于 2017-03-16

3月15日更新了新版本的Shadowsocks后，我惊奇地发现原本自带的kcptun被分离出来了。现在想使用kcptun的功能不只需要下载shadowsocks的kcptun插件，还要重新配置kcptun的参数。

新版的kcptun插件下载地址。
在新版的Shadowsocks配置文件的最下面选中kcptun插件，之后就可以配置参数了。

这次的远程端口配置与上次不一样：

  • 如果不使用kcp插件加速的话，在远程端口处填写ss端口。
  • 如果使用kcp插件进行加速的话，远程端口应设置为kcptun的加速端口。

接下来的kcp参数配置的格式也不同了，现在改成了key=value;的格式。

PS：如果全部配置完成还不能成功连接的话，需要注意，kcptun参数中的key参数的值不需要加引号：key=password;
