hexo+github搭建自己的博客

发表于 2017-03-04 更新于 2017-10-13

准备工作

  • 安装git
  • 安装Node.js
  • 安装hexo
  • 注册github账号

创建博客项目

首先在github上新建一个仓库取名为“xxx.github.io”。其中xxx为github的用户名，比如我的用户名为zinc357，那么仓库名就是xxx.github.io。

接下来在本地创建一个文件夹，作为blog的目录。然后进入gitbash，进入这个文件夹输入命令安装插件

npm install hexo-deployer-git --save

然后输入命令创建blog的基本目录

hexo init

使用sublime text之类的文本编辑器打开根目录下的_config.yml文件。修改deploy的属性。

仓库属性填上刚才创建的仓库的地址。

现在可以试着输入命令在本地启动hexo试一试了。

hexo g
hexo s

然后在浏览器输入网址 localhost：4000 。就可以看到欢迎页面了。

新建博客并部署到github上

输入

hexo n "文章名称"

之后会在source/_post文件夹下新建的个文章名称的md文件。这个md文件是需要自己编写的文章。
博文写完之后，可以执行

hexo g
hexo s

在本地预览一下。

检查无误后，执行

hexo g -d

部署到github上。
接下来只需要访问 xxx.github.io 就可以看到效果了。
